# Explore Education

Monorepo de proyectos educativos interactivos. Cada subcarpeta es un módulo independiente con su propio stack, dependencias y documentación.

## Proyectos

| Carpeta | Descripción | Stack |
|---------|-------------|-------|
| [`solar_system/`](./solar_system) | Cosmos Explorer — módulo interactivo del sistema solar para niños de 8-12 años. Vista 3D realista de cada planeta, lunas con órbita, anillos de Saturno, cross-section del núcleo. | Vite 5 · React 18 · TS · R3F · Zustand |

## Cómo correr un proyecto

```bash
cd <project-folder>
npm install
npm run dev
```

Ver el README de cada subcarpeta para detalles específicos.
